# FIND_DOGS（リポジトリ/プロジェクト/）
 
PHP自作
 

 
# 概要
 
犬の殺処分を減らすために、保護犬や手放さなければならない方と犬を飼いたい人に向けたサイトを作りました。
ゲストユーザと一般ユーザに分け、 それぞれでログインできるようにしています。
 
# 使い方
 
ゲストユーザ
お問い合わせをすることができます。

一般ユーザ

新規投稿、投稿にコメント、ユーザー編集ができるようになっています。
